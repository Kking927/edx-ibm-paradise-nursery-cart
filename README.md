# IBM edX Paradise Nursery Cart Project

A shopping cart application developed as part of an **IBM edX course**.  
This project allows users to browse nursery products, add items to a shopping cart, and view, update, or remove selected items.  
It demonstrates React and JavaScript skills in a real-world e-commerce scenario, with a clean and interactive interface.

---

## Live Demo

[Live Demo Here](https://kking927.github.io/edx-ibm-paradise-nursery-cart/)

---

## Features

- Browse nursery products with images and details  
- Add items to a shopping cart  
- View, update, or remove selected items from the cart  
- Clean and interactive UI for managing product selections  
- Responsive design for desktop and mobile screens

---

## Tech Stack Used

- **React 18** – Frontend library for building interactive UIs  
- **Vite** – Fast development server and build tool  
- **CSS3** – Styling with layouts, grids, and responsive design  
- **JavaScript (ES6)** – Modern syntax and modules

---

## Skills Showcased

- Building a **React e-commerce interface** with components and state management  
- **Dynamic UI updates** with `useState` and `useEffect`  
- **Form handling** and shopping cart logic  
- **Conditional rendering** for cart updates and product display  
- **CSS styling** for layout, responsive design, and interactive UI elements  
- Structuring a **real-world project** for maintainability and scalability
